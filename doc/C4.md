# C4 : Apresentação

O projeto é apresentado através de uma **apresentação em slides**, mais concretamente num pdf à parte, utilizada como apoio à exposição oral do website e das principais decisões técnicas e de design.

A apresentação aborda:
- Enquadramento e objetivos do projeto;
- Estrutura do website (4 páginas HTML);
- Interface e design responsivo (mobile-first);
- Identidade visual;
- Elementos multimédia;
- Integração de dados com XML e JavaScript;
- Validações e testes;
- Publicação final no Netlify.
