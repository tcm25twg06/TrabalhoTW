# Product
 C3 : Produto

## Produto / Website
O produto desenvolvido é um website informativo e educativo dedicado à comunicação visual
e identidade de marca em multimédia. O site apresenta conteúdos teóricos e exemplos
relacionados com branding, estratégia e tecnologias web.

## Instalação
O projeto encontra-se disponível num repositório GitHub e pode ser executado localmente num navegador web moderno.

Para permitir a execução correta das funcionalidades em JavaScript, nomeadamente o carregamento dinâmico de dados a partir de um ficheiro XML, foi utilizado um servidor local, recorrendo à extensão Live Server do Visual Studio Code.

## Utilização
A navegação é realizada através de um menu principal, presente em todas as páginas, que permite ao utilizador aceder às diferentes secções do site:
- Início
- Identidade Visual
- Estratégia
- Multimédia

Cada página aborda um aspeto específico da comunicação de marca.

## Navegação
A interface gráfica foi desenvolvida com foco na simplicidade, clareza e consistência visual.
A organização do layout em secções facilita a leitura e a navegação, tanto em computador como em dispositivos móveis.

# Validação HTML5 e CSS3
validação do código HTML foi realizada através do W3C Markup Validation Service, testando individualmente cada página do website.
Não foram detetados erros de validação.

A validação do código CSS foi realizada utilizando o W3C CSS Validation Service, garantindo conformidade com as normas do CSS3.

## Para quem é
O website destina-se a estudantes e profissionais das áreas de multimédia, design e
comunicação, bem como a qualquer utilizador interessado em compreender como as marcas
se constroem visualmente no meio digital.

### Validação do XML

O ficheiro [dados.xml](../src/dados.xml) referencia o esquema [dados.xsd](../src/dados.xsd) através dos atributos:

- `xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"`
- `xsi:noNamespaceSchemaLocation="dados.xsd"`

Desta forma, o XML pode ser validado segundo as regras definidas no XSD, garantindo que os dados estão bem formados e corretamente estruturados.
