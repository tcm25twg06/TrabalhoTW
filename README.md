# Comunicação Visual e Identidade de Marca em Multimédia

## Tema / Objetivo
Este trabalho tem como objetivo o desenvolvimento de um website dedicado à explicação de como os recursos multimédia contribuem para a comunicação e contrução da identidade visual de uma marca. 
Na atualidade, num contexto digital cada vez mais competitivo, a forma como as marcas se apresentam visualmente influencia a atenção, a confiança e o interesse do público, ajudando a reforçar a mensagem e a diferenciar a marca.
