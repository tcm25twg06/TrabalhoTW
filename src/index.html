Início

Este website explica, de forma simples, como a comunicação visual e os recursos multimédia contribuem para a construção e expressão da identidade de uma marca.

Nota: é um site educativo. Os exemplos e estruturas servem para demonstrar conceitos e requisitos do trabalho.

Recurso útil: Validador W3C.

O que vais encontrar

  • Identidade Visual: elementos (logótipo, tipografia, cor) e coerência.
  • Estratégia: objetivos, público e consistência na comunicação.
  • Multimédia: formatos e dados estruturados (XML) para sites.

Este parágrafo está escondido (exemplo de requisito CSS).
